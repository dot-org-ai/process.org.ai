# Summarize result(s)

> Business-as-Code definition for summarize result(s). Models the process of outlining the major facts and figures of the quality test results in order to provide insights and information.

## Overview

Outlining the major facts and figures of the quality test results in order to provide insights and information. Use charts, tables, statistical test results, and written findings and conclusions in a summary of results.

## Usage

```typescript
import { summarizeResults } from '@headlessly/summarize-results'

const client = summarizeResults()

// Establish measurable quality criteria for result(s)
const result = await client.defineQualityCriteria({
  scope: 'enterprise',
  period: 'Q1-2025'
})

// Perform quality inspection or test for result(s)
const assessment = await client.executeQualityCheck({
  resultId: result.id,
  criteria: 'standard'
})

// Evaluate quality data and identify trends for result(s)
await client.analyzeResults({
  resultId: result.id,
  format: 'detailed',
  recipients: ['stakeholders']
})
```
