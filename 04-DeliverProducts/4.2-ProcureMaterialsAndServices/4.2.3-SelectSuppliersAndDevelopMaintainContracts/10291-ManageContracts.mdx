# Manage contracts

> Business-as-Code definition for manage contracts. Models the supplier-management processes within procurement to ensure materials and services are acquired efficiently and cost-effectively.

## Overview

Keeping contracts up-to-date with routine evaluation. Maintain order and discipline with the contracts in order to avoid any loss of information and mishaps.

## Usage

```typescript
import { manageContracts } from '@headlessly/manage-contracts'

const client = manageContracts()

// Review requirements for procurement
const requirements = await client.reviewRequirements({
  category: 'contracts',
  urgency: 'standard',
  budgetCode: 'PROC-2025'
})

// Evaluate options and execute
const evaluation = await client.evaluateOptions({
  requirementsId: requirements.id,
  criteria: ['cost', 'quality', 'delivery-time']
})
```
