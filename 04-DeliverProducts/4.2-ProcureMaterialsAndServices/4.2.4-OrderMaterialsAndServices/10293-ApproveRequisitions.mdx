# Approve requisitions

> Business-as-Code definition for approve requisitions. Models the ordering processes within procurement to ensure materials and services are acquired efficiently and cost-effectively.

## Overview

Approving requisitions for materials and services. Examine distributor-specific requests, and validate them individually.

## Usage

```typescript
import { approveRequisitions } from '@headlessly/approve-requisitions'

const client = approveRequisitions()

// Review requirements for procurement
const requirements = await client.reviewRequirements({
  category: 'requisitions',
  urgency: 'standard',
  budgetCode: 'PROC-2025'
})

// Evaluate options and execute
const evaluation = await client.evaluateOptions({
  requirementsId: requirements.id,
  criteria: ['cost', 'quality', 'delivery-time']
})
```
