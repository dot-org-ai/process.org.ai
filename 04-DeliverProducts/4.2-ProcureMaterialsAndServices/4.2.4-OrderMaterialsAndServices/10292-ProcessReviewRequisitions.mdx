# Process/Review requisitions

> Business-as-Code definition for process/review requisitions. Models the ordering processes within procurement to ensure materials and services are acquired efficiently and cost-effectively.

## Overview

Handling operations related to processing/reviewing the requisitions. Establish and maintain procedures for the initiation, authorization, and processing of purchase requirements to procure products/services.

## Usage

```typescript
import { processReviewRequisitions } from '@headlessly/process-review-requisitions'

const client = processReviewRequisitions()

// Review requirements for procurement
const requirements = await client.reviewRequirements({
  category: 'requisitions',
  urgency: 'standard',
  budgetCode: 'PROC-2025'
})

// Evaluate options and execute
const evaluation = await client.evaluateOptions({
  requirementsId: requirements.id,
  criteria: ['cost', 'quality', 'delivery-time']
})
```
